# ractive-dynamodb-json-editor

Ractivejs component for editing dynamodb style json object.
Has no other component dependency.

- [x] String data type
- [x] Number data type
- [x] Boolean data type
- [x] Null data type
- [x] Binary data type
- [x] List data type
- [x] Map data type
- [x] StringSet data type
- [x] NumberSet data type
- [x] BinarySet data type
